# VR Mountain Climbing Scenario

A virtual reality mountain climbing experience built in Unity. The project simulates the physical and environmental challenges of ascending a mountain, offering an immersive first-person VR scenario.

## Built With

- Unity (C#)
- ShaderLab / HLSL for custom visual effects
- XR / VR input and interaction systems

## Getting Started

### Prerequisites

- Unity 6.3
- A compatible VR headset and runtime (e.g. OpenXR, SteamVR, or Oculus)

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Saguny/VR-MountainClimbingScenario.git
   ```
2. Open the `MountainClimbTest` folder as a Unity project.
3. Ensure your VR runtime is configured and your headset is connected.
4. Open the main scene and press Play.
